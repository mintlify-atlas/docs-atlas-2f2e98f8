## What is Civ7 Modding Tools?

Civ7 Modding Tools is an open-source TypeScript monorepo that gives you everything you need to build Civilization VII mods with code instead of hand-editing XML files.

## How it works

<Steps>
  <Step title="Install the toolkit">
    Run `bun install` and `bun run link:cli` to install all packages and register the global `civ7` binary.
  </Step>
  <Step title="Pull game data">
    Run `bun run refresh:data` to extract official game resources into `.civ7/outputs/` using the CLI.
  </Step>
  <Step title="Write your mod in TypeScript">
    Import builders from `@mateicanavra/civ7-sdk`, configure your entities, and call `mod.build('./my-mod')`.
  </Step>
  <Step title="Install and play">
    Copy the generated mod folder to your Civilization VII mods directory and launch the game.
  </Step>
</Steps>

<Note>
  This project requires **Node 22.14+** and **Bun**. See [Prerequisites](/installation/prerequisites) before getting started.
</Note>
