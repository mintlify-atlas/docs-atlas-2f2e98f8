---
title: "ConstructibleBuilder"
description: "Create buildings and improvements with yield changes, valid placement rules, and adjacency bonuses."
---

`ConstructibleBuilder` handles both buildings (placed in districts) and improvements (placed on tiles). The type is inferred from the `constructibleType` prefix: `BUILDING_*` produces a building, `IMPROVEMENT_*` produces an improvement.

## Import

```typescript
import {
  ConstructibleBuilder,
  ACTION_GROUP_BUNDLE,
  CONSTRUCTIBLE_TYPE_TAG,
  DISTRICT,
  YIELD,
} from '@mateicanavra/civ7-sdk';
```

## Generated files

Files are written under `/constructibles/<kebab-case-type>/`:

| File | Content | Action group |
|---|---|---|
| `always.xml` | Types, Buildings/Improvements, Constructibles, TypeTags, ValidDistricts, Yields | Always |
| `localization.xml` | EnglishText | Shell + always |
| `icons.xml` | IconDefinitions | Shell + always |

## Example

```typescript
import {
  Mod,
  ConstructibleBuilder,
  ACTION_GROUP_BUNDLE,
  CONSTRUCTIBLE_TYPE_TAG,
  DISTRICT,
  YIELD,
} from '@mateicanavra/civ7-sdk';

const mod = new Mod({ id: 'my-mod', version: '1' });

const granary = new ConstructibleBuilder({
  actionGroupBundle: ACTION_GROUP_BUNDLE.AGE_ANTIQUITY,
  constructible: {
    constructibleType: 'BUILDING_CUSTOM_GRANARY',
  },
  building: {},
  typeTags: [
    CONSTRUCTIBLE_TYPE_TAG.AGELESS,
    CONSTRUCTIBLE_TYPE_TAG.FOOD,
  ],
  constructibleValidDistricts: [
    DISTRICT.URBAN,
    DISTRICT.CITY_CENTER,
  ],
  constructibleYieldChanges: [
    { yieldType: YIELD.FOOD, yieldChange: 4 },
    { yieldType: YIELD.GOLD, yieldChange: 2 },
  ],
  constructibleMaintenances: [
    { yieldType: YIELD.PRODUCTION, amount: 1 },
  ],
  localizations: [{
    name: 'Custom Granary',
    description: 'A granary that provides extra food.',
    tooltip: '+4 Food, +2 Gold per turn.',
  }],
});

mod.add([granary]).build('./my-mod');
```

## Improvement example

```typescript
const farm = new ConstructibleBuilder({
  actionGroupBundle: ACTION_GROUP_BUNDLE.AGE_ANTIQUITY,
  constructible: {
    constructibleType: 'IMPROVEMENT_CUSTOM_FARM',
  },
  improvement: {},
  constructibleValidTerrains: [TERRAIN.GRASS, TERRAIN.PLAINS],
  constructibleYieldChanges: [
    { yieldType: YIELD.FOOD, yieldChange: 2 },
  ],
  localizations: [{ name: 'Custom Farm', description: 'A farm tile improvement.' }],
});
```

<Note>
  The `building` or `improvement` field must be explicitly set to `{}` only if the `constructibleType` prefix would not auto-detect correctly. In most cases, the prefix alone is sufficient.
</Note>
